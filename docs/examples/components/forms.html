Forms Example

This example demonstrates how to use the ZephCode Forms component.

CSS Snippet for Forms

                    
            .zc-form-group {
                margin-bottom: 1rem;
            }
            .zc-form-label {
                display: block;
                font-weight: bold;
                margin-bottom: 0.5rem;
                color: #212529;
            }
            .zc-form-control {
                width: 100%;
                padding: 0.5rem 1rem;
                border: 1px solid #ced4da;
                border-radius: 0.25rem;
                background-color: #fff;
                transition: border-color 0.15s ease-in-out, box-shadow 0.15s ease-in-out;
            }
            .zc-form-control:focus {
                border-color: #80bdff;
                box-shadow: 0 0 0 0.2rem rgba(0, 123, 255, 0.25);
            }
            .zc-form-floating {
                position: relative;
                margin-bottom: 1rem;
            }
            .zc-form-floating .zc-form-control {
                padding-top: 1.625rem;
                padding-bottom: 0.625rem;
            }
            .zc-form-floating label {
                position: absolute;
                top: 0.5rem;
                left: 1rem;
                font-size: 0.875rem;
                transition: all 0.2s ease-in-out;
            }
            .zc-form-floating .zc-form-control:focus + label, .zc-form-floating .zc-form-control:not(:placeholder-shown) + label {
                top: -0.5rem;
                left: 0.75rem;
                font-size: 0.75rem;
                color: #007bff;
            }
