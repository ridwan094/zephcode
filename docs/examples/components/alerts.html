Alerts Example

This example demonstrates how to use the ZephCode Alerts component.

This is a primary alert with an icon.
This is a danger alert with an icon.
This is a success alert with an icon.

CSS Snippet for Alerts


/* Base Alert Styles */
.zc-alert {
    padding: 1rem;
    border: 1px solid transparent;
    border-radius: 0.25rem;
    margin-bottom: 1rem;
    display: flex;
    align-items: center;
    gap: 0.5rem;
    animation: fadeIn 0.3s ease-in-out;
}

/* Dismissible Alerts */
.zc-alert-dismissible {
    position: relative;
    padding-right: 2.5rem;
}

.zc-alert-dismissible .zc-alert-close {
    position: absolute;
    top: 50%;
    right: 1rem;
    transform: translateY(-50%);
    background: none;
    border: none;
    font-size: 1.25rem;
    cursor: pointer;
    opacity: 0.7;
    transition: opacity 0.2s ease-in-out;
}

/* Primary Alert */
.zc-alert-primary {
    background-color: #007bff;
    border-color: #0056b3;
    color: #ffffff;
}

/* Danger Alert */
.zc-alert-danger {
    background-color: #dc3545;
    border-color: #bd2130;
    color: #ffffff;
}

/* Success Alert */
.zc-alert-success {
    background-color: #28a745;
    border-color: #218838;
    color: #ffffff;
}

/* Icon Styles */
.zc-alert-icon {
    font-size: 1.5rem;
    margin-right: 0.5rem;
}

/* Animations */
@keyframes fadeIn {
    from {
        opacity: 0;
        transform: translateY(-10px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}
